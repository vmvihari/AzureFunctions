# Azure Functions .NET Enterprise Bootcamp

Welcome to the Azure Functions .NET Enterprise Bootcamp! In this course, you will build an **Event-Driven Order Fulfillment System** from scratch, progressing from junior-level basics to senior-level enterprise patterns.

## The Goal
You will be implementing the project yourself following the instructions provided in the `docs` folder. By writing the code yourself, you will solidify your understanding of serverless architecture in Azure.

## Curriculum

The course is divided into 6 modules. Navigate to each module's folder in `/docs` to read the concepts and complete the tasks.

*   **[Module 1: The Foundation](./docs/module-1-foundation/README.md)**
    *   Isolated Worker Model, Local Dev, HTTP/Timer Triggers, Dependency Injection, Unit Testing.
*   **[Module 2: Event-Driven Architecture](./docs/module-2-messaging/README.md)**
    *   Service Bus (Queues/Topics), Blob Storage, Cosmos DB bindings, Dead Lettering.
*   **[Module 3: Orchestration and State](./docs/module-3-orchestration/README.md)**
    *   Durable Functions, Fan-out/Fan-in, Sagas, Compensation logic.
*   **[Module 4: Enterprise Security & Resilience](./docs/module-4-enterprise/README.md)**
    *   Managed Identities, Key Vault, Polly (Retries/Circuit Breakers), APIM.
*   **[Module 5: Observability and DevOps](./docs/module-5-devops/README.md)**
    *   Application Insights, Terraform (IaC), Azure DevOps YAML pipelines.
*   **[Module 6: Real-Time & Advanced Production Patterns](./docs/module-6-advanced/README.md)**
    *   SignalR Service, Event Hubs vs Event Grid, KEDA, Premium Plans.

## Getting Started
Head over to `/docs/module-1-foundation/README.md` to begin your journey!
